Effective 1 September 2019, all new OGC standards that reference the "Well Known Text Representation of Coordinate Reference Systems" standard shall reference the latest version of that standard [OGC 18-010r7], also referred to as CRS WKT2. Standards Working Groups are encouraged to revise existing standards to reference CRS WKT2 where possible.
